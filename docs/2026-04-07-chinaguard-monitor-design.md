# ChinaGuard Monitor — Спецификация

## Обзор

**ChinaGuard Monitor** — бот-мониторинг площадок для поиска потенциальных клиентов сервиса ChinaGuard. Находит посты/вопросы по ключевым словам, генерирует экспертные черновики ответов через AI, присылает уведомления в Telegram с кнопками.

**Модель:** полуавтомат — AI генерирует черновик, человек одобряет и публикует вручную (переход по ссылке + вставка текста).

**AI-провайдер:** AITunnel (aitunnel.ru) — русский аналог OpenRouter, оплата в рублях, модель GPT-4o mini.

**Деплой:** на том же VPS что и основной бот ChinaGuard.

**Частота проверки:** каждые 30 минут (кроме Telegram-чатов — в реальном времени).

## Парсеры площадок

### 5 площадок:

| Площадка | Метод | Что парсим |
|----------|-------|-----------|
| vc.ru | RSS-лента + requests + BeautifulSoup | Новые статьи и комментарии по ключевикам |
| Pikabu | HTML-парсинг (requests + BeautifulSoup) | Новые посты по тегам |
| forum.alta.ru | HTML-парсинг новых тем | Заголовки тем в разделе таможенного форума |
| Дзен | HTML-парсинг поиска | Новые статьи по ключевым словам |
| Telegram-чаты | Telethon userbot (реальное время) | Сообщения в чатах по ВЭД |

### Telegram (Telethon):
- Используется личный аккаунт Telegram (userbot, не бот)
- Telethon подключается под номером пользователя и слушает чаты
- Реагирует мгновенно, не каждые 30 мин
- Нужно получить api_id и api_hash на my.telegram.org
- Чаты по ВЭД/Китаю будут найдены и добавлены на этапе реализации

## AI-генерация черновиков

**Провайдер:** AITunnel (aitunnel.ru)
**Модель:** GPT-4o mini
**Эндпоинт:** https://api.aitunnel.ru/v1/chat/completions
**Стоимость:** ~10-30 руб/мес при 20 запросах/день

**Системный промпт:**

```
Ты — эксперт по китайскому праву и ВЭД. Ты помогаешь 
российским предпринимателям решать проблемы с китайскими 
партнёрами.

Человек написал на форуме/в чате:
"{текст_поста}"

Напиши экспертный комментарий (3-5 предложений):
1. Кратко квалифицируй ситуацию по ГК КНР (укажи конкретную статью)
2. Дай практическую рекомендацию
3. Заверши фразой: "Для детальной проверки вашей ситуации — @chinaguard_bot"

Тон: профессиональный, уверенный, без воды. 
Не используй эмодзи. Пиши на русском.
```

## Telegram-бот уведомлений

**Отдельный бот** (не ChinaGuard бот). Только для админа.

### Формат уведомления:

```
📌 vc.ru — новый пост
━━━━━━━━━━━━━━━━━━

"Заказали партию электроники в Шэньчжэне, 
оплатили 30% предоплаты. Прошло 2 месяца — 
поставщик перестал отвечать..."

🔗 https://vc.ru/trade/123456

💬 Черновик ответа:
По ст. 577 ГК КНР поставщик, не исполнивший 
обязательства, несёт ответственность за нарушение 
договора...
Для детальной проверки — @chinaguard_bot

[📋 Скопировать ответ] [⏭ Пропустить]
```

### Кнопки:
- **Скопировать ответ** — копирует черновик (callback, бот отправляет текст отдельным сообщением для удобного копирования)
- **Пропустить** — помечает пост как обработанный

### Команды:
- `/stats` — сколько постов найдено за день/неделю/месяц
- `/pause` — приостановить мониторинг
- `/resume` — возобновить
- `/keywords` — показать текущие ключевые слова

## База данных (SQLite)

**Таблица posts:**

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PK | Автоинкремент |
| platform | TEXT | Название площадки (vc, pikabu, alta, dzen, telegram) |
| post_url | TEXT UNIQUE | URL или ID поста (для дедупликации) |
| title | TEXT | Заголовок/начало текста |
| matched_keyword | TEXT | Какое ключевое слово сработало |
| draft_response | TEXT | Сгенерированный черновик |
| status | TEXT | new / sent / copied / skipped |
| created_at | TIMESTAMP | Когда найдено |
